# FinnHubTracker

### Description

A Flutter mobile app that monitors current prices of finnHub stocks prices.



### Setup


1. **Clone the Repository:**

    ```bash
    git clone https://github.com/carlosred/finnHubTracker.git
    cd finnHubTracker
    ```

2. **Install Dependencies:**

    ```bash
    flutter pub get
    ```

### Running the App

#### Android

```bash
flutter run -d android
```
